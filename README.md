# Nayan Pokhriyal Portfolio

Interactive portfolio built with Next.js, TypeScript, Tailwind CSS, and custom glass/particle interactions.

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment

Create `.env.local` from `.env.example`:

```bash
GITHUB_TOKEN=your_github_token
```

The token is used server-side for GitHub contribution data. Do not commit `.env.local`.

## Scripts

```bash
npm run lint
npm run build
```
